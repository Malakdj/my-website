Atbah

Definition

The Atbah Cipher is a simple substitution cipher where the alphabet is reversed. Each letter in the plaintext is replaced by its opposite letter from the alphabet (A ↔ Z, B ↔ Y, C ↔ X, etc.). It is a monoalphabetic cipher, meaning each letter is substituted with another letter consistently throughout the message.

How the Atbah Cipher Works

The Atbah Cipher is a straightforward encryption method that operates by mapping each letter of the alphabet to its reverse counterpart.

Encryption Process

  1. Write the alphabet from A to Z.
  2. Write the alphabet from Z to A (reverse order).
  3. For each letter in the plaintext, replace it with the letter at the same position in the reverse alphabet.

Decryption Process

  1. For each letter in the ciphertext, replace it with the letter at the same position in the normal alphabet.

Example

If A is replaced by Z, B by Y, C by X, and so on:

Plaintext: HELLO → Ciphertext: SVOOL

Weaknesses

  • Vulnerable to frequency analysis because it is a monoalphabetic cipher.
  • Simple to break if the message length is long enough.
